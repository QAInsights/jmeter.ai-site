# Welcome to JMeter.AI

JMeter.AI is a platform for AI-powered performance testing tools and plugins for Apache JMeter. JMeter is one of the most popular open-source performance testing tools, and JMeter.AI aims to make it even more powerful with AI.


## What's in JMeter.AI?

### Feather Wand

Feather Wand is an AI-powered chat interface for Apache JMeter. It allows you to generate JMeter test plans using natural language descriptions, debug, analyze, and more. It has even integration with Claude Code for enhanced AI capabilities.

### JMeter MCP

JMeter MCP helps you integrate JMeter with AI models through the Model Context Protocol (MCP). It allows you to use AI models to enhance your performance testing workflows.

### Super Key

Super Key is a JMeter command center where you can do almost anything from the keyboard. No more right clicks in JMeter.

### JMeter Studio

Can JMeter look this good? Bring vibes to JMeter using the themes and custom icons. One of the fastest downloads in the JMeter Plugins repository.

### Prism

Multi tabs in JMeter so that you can multitask.

### Plugins.JMeter.AI

A modern JMeter Plugins directory for future AI-ready plugins.

### Academy

Don't vibe code or test. Vibe learn, start from scratch, and build your skills.

Join our community to get help, share your experiences, and stay updated with the latest news and updates.
